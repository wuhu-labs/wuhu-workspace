# Wuhu Workspace

This is the Wuhu workspace — the knowledge base for the project.

- `issues/` — Issue tracking (WUHU-#### format)
- `docs/` — Vision, architecture, and design documents
- `docs/vision/` — Product vision and strategy
- `docs/architecture/` — Technical architecture and package design
- `docs/projects/` — Project-level plans and milestones
