# WUHU-0051: Trim tool surface

## Summary

Reduce the agent tool count from 21 to ~12 by removing tools that are
unprincipled, redundant, or better served by a future JS repl.

## Tools to remove

| Tool | Reason |
|------|--------|
| `swift` | Bizarre as an agent tool. Will be replaced by JS repl. |
| `fork` | Channel-specific, removed with channels (WUHU-0050). |
| `env_create` | Renamed and reduced â€” see below. |
| `env_update` | Full CRUD over tool calls is overkill. |
| `env_delete` | Full CRUD over tool calls is overkill. |
